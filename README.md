# macos-wifi-scripts
A collection of scripts to troubleshoot WiFi on MacOS. 
The intention is that the scripts can work by themselves or as a group via the wifi-menu.sh.

| MacOS Wi-Fi Network Adjuster Tools Menu                                     |
